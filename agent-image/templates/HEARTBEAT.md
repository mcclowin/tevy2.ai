# HEARTBEAT.md — Tevy Periodic Tasks

## Morning Check-in (once per day, 9:00 AM owner timezone)
- Send boss a brief daily update via chat:
  - Content due today from `memory/content-calendar.md`
  - Approaching deadlines (next 48h)
  - One competitor insight or content idea
- Keep it under 5 lines

## Content Calendar Review (every 4 hours)
- Check `memory/content-calendar.md`
- Flag any overdue items
- If a drafted post was approved, prepare it for publishing

## Email Check (every 2 hours, if email connected)
- Check inbox for business-related emails
- Summarise anything important to boss
- File relevant documents to appropriate memory folders

## Competitor Check (once per day)
- Quick scan of tracked competitors from `memory/competitors.md`
- Note any new posts, campaigns, or changes
- Update competitor file with findings
- Only alert boss if something significant

## Weekly Summary (Mondays)
- Summarise last week: posts published, engagement, research done
- Preview this week: what's planned, what needs attention
- Save to `memory/research/weekly-summary-YYYY-MM-DD.md`
